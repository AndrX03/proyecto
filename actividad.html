programacion

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea. Pero no solo la acción de escribir un código para que la computadora o el software lo ejecute, por lo que tenemos que valorar su importancia actualmente

importancia de la programacion

No es algo nuevo decir que en la actualidad vivimos en un mundo totalmente virtual y la programación y estudio de la tecnología viene siendo necesario para la sociedad actual dándonos comodidades que hace años nadie ni siquiera imaginaba, y como seres humanos buscamos la evolución y dado que estos temas informáticos vinieron para quedarse debemos de ver la programación como la puerta que no lleva a innovar y crear nuevos programas que nos ayuden como sociedadLa programación es fundamental para acrecentar el avance tecnológico dentro de las industrias que para llevar a cabo sus funciones necesitan de sitios y aplicaciones creadas a partir de los códigos. No es novedad que vivimos en un mundo computarizado donde la tecnología forma parte de todo lo que hacemos.

Para que sirve cada lenguaje?

A la hora de desarrollar aplicaciones, paginas web o lo que sea que tengas en mente debes de saber con que lenguaje trabajaras ya que cada uno tiene sus usos, fortalezas y limitaciones por lo que no puedes desarrollar una aplicacion en ios con css o usas html puro para programar un juego por eso veremos diferentes lenguajes de programacion y sus usos comunes...

  • html:

    html lo veremos luego mas a profundidad pero este lenguaje es usado comunmente para el desarrollo de paginas webs, por lo que si te interesa el desarrollo web es de suma importancia que le vayas echando un ojo

  • css:

    css tiene la funcion de editar colores y modificar aspectos visuales de html por lo que junto a este puede ayudar a crear una pagina web en condiciones

  • javascript:

    JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. (Está bien, no todo, pero es sorprendente lo que puedes lograr con unas pocas líneas de código JavaScript), ademas de ser un lenguaje relativamente facil de aprender es perfecto para inciar con este mundo de la programacion

  • java

    Java es un lenguaje que te servira para el desarrollo de apps en android aunque ultimamente a dejado de ser el lenguaje predilecto para esta tarea ya que an llegado nuevos lenguajes con una mejor curva de aprendisaje que pueden remplazarlo facilmente, java es un lenguaje muy extricto en cuanto a su sintaxis por lo que no lo considero un buen lenguaje para iniciar con la programacion pues puede que te artes y dejes este mundo por esas problematicas, si llegas a dominar este lenguaje seras capaz de crear programas de aclculo o de utilidad ademas de juegos para android o aplicacion, sin duda un gran reto con una gran recompenza

    CSS

    CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas. se suele usar junto a css para dar una mejor experiencia al usuario final

    Etiquetas

    etiquetas funcion Ejemplo
    color la etiqueta color funciona en base al rgb que como sus siglas indican rojo azul y verde cobinando estos para dar tu color color: rgb(84, 250, 84);
    width este modifica que tan grande es algo y puedes expresarselo en pixeles, porcentaje, etc width: 100px;
    margin basicamente una margen, puedes elegir que tamaño y a que lado ira la margen margin: 5px;
    border es como la margen pero en ves de crear un espacio exterior este crea uno interno, se le puede cambiar el color, tamaño y estilo border: solid ipx red
    font-family cambia la fuente de la letra font-family: fuente;
    background modifica el fondo, actualmente desconozco mas utilidades aparte del color background-color: green;
    float hace flotar una seccion lo que hace que traspase aun no la entiendo muy bien por lo que actualizare esto -proximamente- float: left;
